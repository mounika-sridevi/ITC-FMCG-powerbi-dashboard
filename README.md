# ITC FMCG Sales Dashboard – Power BI

This repository contains a Power BI dashboard analyzing ITC FMCG sales performance.

## Files

- `ITC_FMCG_Sales_Dashboard.pbix` – Main Power BI report
- `data_itc_sales.xlsx` – Underlying sales dataset (sample / dummy data)
- `Page1_Summary.png` – Screenshot of the summary dashboard
- `Page2_RegionWiseSales.png` – Region-wise performance view
- `Dashboard_Documentation.pdf` – Methodology, KPIs, and insights

## How to Open

1. Download the `.pbix` file.
2. Open it in **Power BI Desktop**.
3. Refresh data if needed and explore the visuals.

## KPIs Included

- Total Sales
- Profit Margin
- Region-wise Sales
- Top 10 Products by Revenue
- Month-on-Month Growth
